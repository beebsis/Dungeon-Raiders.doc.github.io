Get started

In order to get started you'll have to make a character by typing !join class. You can pick between 5 classes, farmer, rogue, warrior, ranger and wizard. Each class has their pros and cons. We recommend trying each one out for a bit. You can read more about the different classes and their utility in the game on the Character section.

Command: !join class

Skills

Each Character has 3 skills, you might want to change skill depending on the situation, composition of for testing. There are both defense, offense and utility abilities in this game. Use them according to the situation.

Command: !setskill #

Relics

Auras

Auras are an extra stat bonus that will increase without limit as you progress. They are obtained from Guardians in the Underworld, and everyone that enters the portal will get them, no matter if they survived the floor where it got dropped or not. You can project your Aura at level 200+ and you can only have 1 active Aura at a time. You need some Aura before you can use it, so join a Portal run, but for that you need to be at level 200 or higher. Not every aura will become available during a Portal raid, so you may need to join more portal raids before you can activate the one you want to use.

Command:!setaura # - Set your aura

Requres: Hero+ status

Command:!setraidaura # - Set an aura for raiding/guarding

Command:!setmineaura # - Set an aura for mining

Command:!settrainaura # - Sets an aura for traning

List of auras:

1.) Vitality Increases Maximum hitpoints and healing
2.) Wisdom Increases experiance gain
3.) Braveness Damange and critical chance increase
4.) Abundance Gold & farming rate increase

Buildings

Upgrading a building will give you benefits on that particular building and also increase your town maximum HP. A few buildings have two numbers next to them. The (x/y) indicates how many players are using the building (x) and the (y) the maximum number that that can be present at the same time

Command: !upg (building name) - Upgrades the building

Command: !voteyes - Votes yes for building upgrade

Command: !voteno - Votes no for building upgrade

Command: !buy# - Purchase shop item

Command: !farm - Farms food for the town

Command: !mine - Mines for gold and gems

Command: !enchant# - Upgrading your relic

Command: !sacrifice# - Sacrifice souls to start the witch

Command: !forge # - Forges / upgrades equipment

Command: !reforge# - Crafts a relic of choice

Command: !recycle# - Sells relic for gems

Command: !sellgem# - Sells X amount of gems for 1m each

Command: !chest# - Choose a chest
