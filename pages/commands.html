Get started

In order to get started you'll have to make a character by typing !join class. You can pick between 5 classes currently (21-09-2021), farmer, rogue, warrior, ranger and wizard. Each class has their pros and cons. We recommend trying each one out for a bit. You can read more about the different classes and their utility in the game on the Character section.

Command: !join class

Usable when: No joined character

Skills

Each Character has 3 skills, you might want to change skill depending on the situation, composition of for testing. There are both defense, offense and utility abilities in this game. Use them according to the situation.

Command: !setskill #

Usable when: In town
